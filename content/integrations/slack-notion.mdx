# Slack and Notion

Use Slack and Notion together to turn discussion into durable knowledge and action.

## Common Agent Capabilities

- Post summaries and alerts to Slack channels
- Read thread context before responding
- Save decisions and runbooks to Notion
- Keep team docs aligned with operational updates

## Setup Flow

1. Open [studio.scoutos.com/integrations](https://studio.scoutos.com/integrations)
2. Connect **Slack** and **Notion**
3. Enable tools for the specific agent
4. Define channel and page scope in instructions

## Instruction Snippet

```markdown
For Slack and Notion actions:

1. Read channel or page context before posting or updating.
2. Keep Slack posts concise and action-oriented.
3. Write durable details to Notion, not only Slack.
4. Return links to the Slack message and Notion page after writes.
```

## Prompt Examples

- "Summarize this incident thread and post action items to #ops."
- "Create a Notion page for this project brief and link it in Slack."
- "Read today's standup messages and produce a risk summary."
