# Email and Calendar

Use email and calendar integrations for daily execution workflows: triage, scheduling, prep and follow-ups.

## Common Agent Capabilities

- Triage inbound mail by priority or topic
- Draft replies with context from CRM and docs
- Check availability and propose meeting times
- Generate prep briefs before scheduled calls

## Setup Flow

1. Open [studio.scoutos.com/integrations](https://studio.scoutos.com/integrations)
2. Connect your email and calendar accounts
3. Enable related tools in the agent's **Tools** tab
4. Add communication guardrails in instructions

## Instruction Snippet

```markdown
For email and calendar actions:

1. Draft first, send second.
2. Confirm recipient, subject and timezone before sending or booking.
3. Include a concise rationale for each scheduling decision.
4. Return message IDs and event IDs after actions.
```

## Prompt Examples

- "Summarize my unread high-priority emails and draft replies."
- "Find two meeting windows next week for this group in Pacific time."
- "Create a prep brief for tomorrow's customer call from recent email context."
