# Welcome to Scout

Scout is an **agentic workforce studio**: build, deploy and manage autonomous AI agents that get real work done.

## Build Your Digital Workforce

Whether you're a developer who loves code or a business operator who prefers no-code tools, Scout gives you the power to create intelligent agents that work independently:

- **No-code builders** — Use Scout Studio's visual interface to configure agents with natural language
- **Developers** — Use our Python and TypeScript SDKs for programmatic control
- **Everyone** — Access the Agent Marketplace for ready-to-deploy templates

Same powerful agents. Your choice of how to build.

## Key Capabilities

| Feature | What It Does |
|---------|--------------|
| **Agent Marketplace** | Start fast with pre-built agent templates for common business tasks |
| **Scheduling** | Run agents automatically on schedules, no workflows required |
| **Planning Tools** | Give agents goals in plain English and they figure out the path |
| **Drive** | Built-in file storage for documents, images and assets |
| **Collections** | Vector databases for semantic search and knowledge retrieval |

## What Can Your Agents Do?

- **Customer Support** — Handle tickets, look up history, escalate urgent issues
- **Research & Analysis** — Gather competitive intel, summarize findings, create reports
- **Document Processing** — Extract data from PDFs, update systems, automate paperwork
- **Knowledge Assistance** — Answer questions from your docs, wikis and databases
- **Workflow Automation** — Connect your tools and automate multi-step processes
