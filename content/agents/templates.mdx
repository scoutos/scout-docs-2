# Agent Templates

Pre-built agents you can install and customize in minutes. The Agent Marketplace is the fastest way to get started with Scout.

## What are Agent Templates?

Agent templates are production-ready agents designed for common business tasks. Each template comes with:

- **Pre-written instructions** - Optimized prompts for the specific use case
- **Connected tools** - The right integrations already selected
- **Suggested settings** - Model selection, step limits, and response styles tuned for the task
- **Guided setup** - Walks you through connecting your accounts

Instead of spending hours configuring an agent, install a template and start getting value immediately.

## Available Templates

### Meeting Prep

Prepare for any call with comprehensive company research and key insights.

**What it does:**
- Researches the company before your meeting
- Pulls recent news and announcements
- Identifies key decision-makers and their backgrounds
- Summarizes relevant CRM activity and previous interactions
- Generates a briefing document with talking points and questions to ask

**Best for:** Sales reps, account managers, executives preparing for client calls

**Integrations:** Salesforce, HubSpot, Web Search, LinkedIn (optional)

**Sample prompt:**
> "I have a call with Acme Corp tomorrow at 2pm. VP of Sales is joining. Prep me."

## Installing a Template

### Step 1: Browse the Marketplace

Navigate to [studio.scoutos.com/agents/marketplace](https://studio.scoutos.com/agents/marketplace)

### Step 2: Select a Template

Click on any template to see:
- What it does
- Required integrations
- Sample prompts

### Step 3: Click Install

The guided setup will:
1. Create your agent with pre-configured instructions
2. Prompt you to connect required integrations (if not already connected)
3. Set up suggested schedules (if applicable)

### Step 4: Start Using It

Once installed, click **Chat** to start working with your agent. You can also:

- **Modify instructions** - Customize the behavior
- **Add or remove tools** - Change what it can access
- **Adjust settings** - Fine-tune model and response style
- **Set schedules** - Automate regular tasks

## Customizing Templates

Templates are starting points, not straitjackets. After installation, you have full control:

### Modify Instructions

Tweak the system prompt to match your specific needs:

```
Original: "Research companies before meetings..."
Customized: "Research SaaS companies before meetings, focusing on 
recent funding rounds and product announcements..."
```

### Add or Remove Tools

Connect additional integrations or remove access to tools you don't need.

### Create Variations

Install the same template multiple times with different configurations:
- One Meeting Prep agent for enterprise accounts
- Another for mid-market with different research depth

### Share with Your Team

Set visibility to **Team** so everyone can use the agent.

## Creating Custom Templates

Have an agent that works well for your team? You can create your own templates:

1. Build and refine your agent
2. Go to agent settings
3. Click **Create Template**
4. Add description and sample prompts
5. Share with your workspace

This helps standardize best practices across your organization.

## Frequently Asked Questions

**Can I install multiple templates?**

Yes! Install as many as you need. Each becomes a separate agent you can customize independently.

**What if a template doesn't match my exact use case?**

Templates are fully customizable after installation. Adjust instructions, tools, and settings to fit your needs.

**Do I need to have integrations ready beforehand?**

No. The guided setup will prompt you to connect accounts during installation. You can also skip some and add them later.

**Can I see the instructions before installing?**

Yes. Click on any template in the marketplace to see its configuration before installing.

**Are templates free?**

Templates are free. You only pay for agent usage (model calls, tool executions) based on your Scout plan.
