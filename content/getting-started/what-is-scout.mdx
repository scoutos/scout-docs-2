# What is Scout?

Scout is an **agentic workforce studio** — a platform where anyone can create, deploy, and manage autonomous AI agents that get real work done.

## Build Your Digital Workforce

Whether you write code or prefer no-code tools, Scout gives you what you need to build agents that work autonomously:

- **For technical users** — Full API access, SDKs (Python & TypeScript), and the ability to run agents programmatically
- **For non-technical users** — Visual workflow builder, natural language configuration, and a friendly studio interface

Same agents. Different paths. You choose how to build.

## What Makes Scout Different?

### 🎯 Goals Over Scripts

Traditional automation breaks when something unexpected happens. Scout agents adapt. Give them a goal in plain English, and they figure out the path.

**Old way:** "If email subject contains 'urgent', forward to support@..."

**Scout way:** "Handle all incoming support emails and escalate the ones that need immediate attention"

Agents handle edge cases, try different approaches, and recover from errors automatically.

### 🔌 Connect Everything, Configure Nothing

Stop wrestling with API keys and OAuth flows. Scout handles authentication for you:

- One-click connections to Slack, Google Workspace, Notion, CRMs, databases, and more
- Secure credential management
- Automatic token refresh

### 🛠️ No-Code + Code: You Get Both

Build with **Scout Studio** — our visual workflow builder — or write code with our SDKs. Mix and match as needed:

- Start in the visual builder, export to code when you're ready
- Build custom tools visually, use them in code
- Prototype in the studio, deploy via API

### 💾 Built-in Storage & Knowledge

Your agents need memory. Scout includes it:

- **Drive** — File storage for documents, images, and assets
- **Collections** — Vector databases for semantic search and RAG applications

No external infrastructure required.

Intelligent autonomous workers that connect to your tools, understand natural language instructions, and execute complex tasks. They can collaborate, handle errors gracefully, and adapt to unexpected situations.

## Real-World Use Cases

### Customer Support Automation

*"Handle incoming support tickets, look up customer history, and escalate urgent issues to the right team."*

Connects to email, Slack, your CRM, and internal knowledge base.

### Research & Analysis

*"Research our top 5 competitors and create a pricing comparison report."*

Searches the web, extracts pricing pages, synthesizes findings into a structured report.

### Document Processing

*"Process all incoming invoices, extract key data, and update our accounting system."*

Reads PDFs, extracts line items, pushes data to QuickBooks or your ERP.

### Knowledge Assistant

*"Answer employee questions about our policies and procedures."*

Connects to your Notion wiki, Google Drive, and internal docs for instant answers.
