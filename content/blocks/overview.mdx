# Blocks Overview

Blocks are the building blocks of Scout workflows.

## What is a Block?

A block is a single unit of functionality in a workflow. It receives input, processes it, and produces output.

## Block Categories

- **Input/Output** - Handle data entry and exit
- **AI Processing** - LLM and AI-powered operations
- **Control Flow** - Direct workflow logic
- **Communication** - Send messages and notifications
